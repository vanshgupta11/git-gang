About the Project

LeetCode Automater is a powerful Node.js-based automation tool designed to streamline your LeetCode practice workflow. This project helps you:

  • Automatically fetch your solved LeetCode problems using GraphQL queries
  • Submit Java solutions to LeetCode using your authenticated session
  • Skip premium-only and database-only questions intelligently
  • Track your submission progress with detailed logging
  • Extract LeetCode tokens easily with the Chrome extension

Note: A standalone .exe file is not currently available. Please follow these steps to run the project from source.
